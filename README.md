### 🏭 Jalaram Distributors — Wholesale Billing & ERP System

> *Production software running a live wholesale distribution business in Gondal, Gujarat — every single day.*

<br>

This is not a tutorial project — it's a **real production ERP** that handles daily orders, billing, inventory, and payments for an actual wholesale distribution business. Built from scratch and actively used by 6–10 salesmen in the field.

**What makes this production-grade:**

| Feature | Details |
|---|---|
| 🔄 **Real-time Orders** | WebSocket live sync — order placed by salesman instantly appears on shop PC |
| 📵 **Offline PWA** | Salesmen take orders without internet; IndexedDB queue auto-syncs when back online |
| 🔐 **Auth System** | PIN login + JWT + Refresh tokens with 30-day remember-me and device session manager |
| 🖨️ **Challan Printing** | A4 J.D. challan with amount in words (Indian system), round-off, disc/CD columns |
| 📍 **GPS Tracking** | Each order geo-tagged; live salesman location with route history on Leaflet maps |
| ☁️ **Cloud Backup** | Nightly Google Drive backup via Vercel Cron with 30-day retention and failure alerts |
| 🏪 **Area Management** | Routes, streets, salesman assignments — shops auto-appear under their area |
| 📦 **Inventory Control** | Atomic stock deduction with rollback on cancellation; low-stock WebSocket alerts |
| 💳 **Collections** | Outstanding dues tracker, payment recording (Cash/UPI/Cheque), customer ledger |
| 🔒 **Security** | Helmet, CORS, rate limiting, soft delete, audit trail, idempotency keys |

**Tech Stack:**
`React` `Vite` `Tailwind CSS` `Framer Motion` `Node.js` `Express` `MongoDB Atlas` `Socket.IO` `JWT` `PWA` `Leaflet` `Vercel`
